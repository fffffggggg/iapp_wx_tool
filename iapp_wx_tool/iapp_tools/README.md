#### 平台功能说明
------------------------

> 1. 身份证归属地查询
> 2. 银行卡归属地查询
> 3. 手机号码归属地查询
> 4. 标准身材计算
> 5. 小游戏（掷色子）
> 6. 实况天气，未来三天天气
> 7. IP地址查询
> 8. 2048小游戏
> 9. 更多功能还在增加中
